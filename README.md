# My Own Training

## Sobre o projeto

O **My Own Training** é um marketplace que conecta alunos e profissionais de Educação Física de forma simples, rápida e segura.

A plataforma utiliza geolocalização para aproximar usuários de especialistas da sua região, permitindo comparar modalidades, visualizar avaliações, consultar valores e agendar treinos com total transparência.

Para os profissionais, oferecemos ferramentas de gestão de agenda, recebimento de pagamentos, programas de fidelização e recursos de destaque para ampliar a visibilidade e impulsionar sua carreira.

Nosso objetivo é transformar a busca por um treinador pessoal em uma experiência tão prática quanto solicitar um serviço por aplicativo.

---

## Funcionalidades

- 🔍 Busca de profissionais por geolocalização
- 📋 Perfis com avaliações, modalidades e preços
- 📅 Agendamento de sessões em tempo real
- 💳 Pagamentos integrados e comissões automatizadas
- 📊 Gestão de agenda e relacionamento com clientes
- 🎯 Programas de fidelização
- 🚀 Recursos premium para maior visibilidade

## Como Executar Localmente

### 1. Clone o repositório

```bash
git clone https://github.com/seu-usuario/my-own-training.git
cd my-own-training
```

2. Instale as dependências
```bash
npm install
```
3. Configure as variáveis de ambiente
Em desenvolvimento

4. Inicie o servidor de desenvolvimento
```bash
npm run dev
```
5. Acesse a aplicação

Após iniciar o servidor, acesse a URL exibida no terminal, normalmente:
```bash
http://localhost:5173
```
